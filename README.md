# Face Recognition Using PCA and ANN

This project implements a Face Recognition System using Principal Component Analysis (PCA) and Artificial Neural Networks (ANN). The system detects human faces, extracts important facial features using PCA (Eigenfaces), and classifies individuals using a trained ANN model.

## Features

* Face Detection using OpenCV
* Feature Extraction using PCA
* Face Classification using ANN
* Real-Time Face Recognition
* User-Friendly Interface
* High Recognition Accuracy

## Technologies Used

* Python
* OpenCV
* NumPy
* Scikit-Learn
* TensorFlow/Keras
* HTML, CSS, JavaScript

## Project Workflow

1. Dataset Collection
2. Image Preprocessing
3. Face Detection
4. PCA Feature Extraction
5. ANN Model Training
6. Face Recognition and Classification

## Applications

* Smart Attendance Systems
* Access Control Systems
* Security and Surveillance
* Identity Verification
